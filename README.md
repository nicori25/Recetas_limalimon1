# LimaLimón

LimaLimón es una aplicación web de recetas orientada a personas con intolerancias alimentarias y preferencias específicas.

## Funcionalidades

- Registro e inicio de sesión
- Gestión de perfil
- Agregar recetas
- Eliminar recetas
- Sistema de favoritos
- Opiniones en recetas
- Valoración con estrellas
- Ruleta de recetas según la temporada

## Tecnologías utilizadas

- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
